# Video App Frontend

React Native / Expo application for browsing and playing videos.

## Setup

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Configuration**:
   Update `API_BASE` in the screen files (e.g., `DashboardScreen.js`, `VideoPlayerScreen.js`) to point to your backend URL.

## Running the App

```bash
npx expo start
```

- Press `i` for iOS simulator.
- Press `a` for Android emulator.
- Use the Expo Go app on your physical device by scanning the QR code.

## Features

- User Signup/Login
- Video Dashboard with thumbnails
- Custom Video Player with backend-authenticated streaming
- Profile settings and logout
